# NFT-Marketplace-backend

## A platform where users can create digital collectibles (NFTs), list them for sale, buy them from others, and track ownership history

## Tech Stack :

- Python
- Fast API
- SQL Alchemy ORM
- PostgreSQL
- uv 
- uvicorn

---

### All the dependencies are in file
pyproject.toml 

---

### To install the dependencies run :
pip install uv
uv sync

---

### Start the backend project with running in the terminal :
uv run main.py
